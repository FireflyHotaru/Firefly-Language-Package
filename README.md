# Firefly-Language-Package
本仓库用于收录、存放**未通过Pull Request合并至模组本体**的简体中文语言文件。

# 📥 下载
**本仓库不在 Releases 发布固定版本，请前往 Actions 获取最新汉化资源包。**
1. 打开上方 [Actions](../../actions) 页面
2. 选择最近一次成功的构建
3. 在下方 Artifacts 处下载 `Firefly-Language-Package`
4. 下载后直接放入 `resourcepacks` 文件夹使用

## 🔧 若文件过期怎么办
由于 Artifact 有时效限制，若下载失效：
1. Fork 本仓库到你的账号下
2. 进入你仓库的 Actions → 允许运行工作流
3. 点击 `Run workflow` 即可一键自行构建

# 🔍 汉化漏键查找工具
https://tool4dev.net/comparison/text-diff

使用方法：
- 左侧：粘贴/放入 en_us.json
- 右侧：粘贴/放入 zh_cn.json
- 翻译键的删除线处即为**未汉化或需要校对的内容**
